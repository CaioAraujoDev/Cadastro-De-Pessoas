# Calculadora em Java

Projeto de cadastro de pessoas simples em Java, utilizando POO.

## Status
Concluído

## Funcionalidades
- Adiciona novas pessoas com nomes, idades e CPFs
- Altera informações das pessoas
- Consulta as informações das pessoas
- Trata erros de entrada inválida

## Tecnologias utilizadas
- Java
- IntelliJ IDEA
- Git e Github

## Como executar
1. Instale o Java(versão 17 ou superior)
2. Clone ou baixe este repositório:  
git clone https://github.com/CaioAraujoDev/Cadastro-De-Pessoas.git
3. Abra o projeto no seu IDE(recomendado: Intellij IDEA)
4. Execute a classe principal(Main)
